$2000-$2FFF  Character Set RAM

$D000-$D00F  Sprite Positions

$D100-$D1FF  Raster Effect Registers

$D800-$DBE7  Color RAM

$DC00         Joystick Port 2

$DC01         Joystick Port 1

$FF00         Frame Counter

$FF01         VSync Flag
